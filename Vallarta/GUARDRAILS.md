# GUARDRAILS
This file defines safety and operational constraints for development in this workspace.

## Immutable Constraints
- **Scope Discipline**: Make only the changes requested. Do not make off-scope changes.
- **Strict Verification**: Always verify changes build and/or pass linting.
- **Design Excellence**: Implement gorgeous visual styling following `emil-design-eng` and `high-end-visual-design` principles.
